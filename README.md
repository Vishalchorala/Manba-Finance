Manba Finance is a fully responsive web application built using React.js and JSX, designed to provide users with a seamless experience in managing various loan services, including home loans, business loans, gold loans, and personal loans. This project features a clean and modern design, enhanced with Bootstrap for styling and responsiveness across all devices.

Key Features:

Responsive Design: Optimized for all device sizes, ensuring a great user experience on desktops, tablets, and mobile devices.
Page Components: Comprehensive pages including Home, Contact, and Service sections.
Loan Services: Dedicated pages for Home Loan, Business Loan, Gold Loan, and Personal Loan information.
React Router: Implemented to enable smooth navigation between pages.
SVG Icons: Utilized for an appealing and professional interface.
React Packages: Integrated various React packages to enhance functionality and user experience.
